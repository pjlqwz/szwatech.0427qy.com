喜报！深圳龙华红木山水厂运管数字化解决方案入选2023年智慧水务典型案例

2024-01-02

近日，住房和城乡建设部科技与产业化发展中心发布2023年度智慧水务典型案例，“深圳市龙华区红木山水厂运管数字化解决方案”经过形式审查、初审及专家评审等环节，从全国101个申报案例中脱颖而出，最终入选为19项智慧水务典型案例之一，成为饮用水安全保障类的优秀案例。




深圳市龙华区红木山水厂运管数字化解决方案是深圳龙华水务公司与深圳水务科技公司的集体智慧结晶。该方案基于深圳龙华水务公司实际，以安全可靠、稳定高效为目标，解决了运行过程中的痛点和难点，大力推动供水数字化建设，深化数据智能应用，促进水生产运营控制、管理和决策更加智能化和精细化。


在“高级自控+智慧厂站平台+专家系统”新模式下，红木山水厂生产运行稳定性和可靠性得到了大幅提升，水质更有保障，设备运行更高效，实现了现场无人值守辅以低频巡检的运管新模式。


智慧水厂平台综合界面


// 高级自控

红木山水厂在自控方面做了全工艺段升级，实现了全流程工艺自动化、取水泵房无人值守、智能加药、三网分离、全厂自控网络覆盖。正是全工艺自动化控制，才有了全厂的高效运行。



// 智慧厂站平台

红木山水厂智慧厂站平台是基于数字孪生技术，实现了在线视频巡检，打破了空间限制，实时掌握了工单动态及完成情况，实时监测到CCP点，对生产、运营、管理、水质一目了然，形成全过程闭环管控。




// 专家系统

红木山水厂在设备预诊断、配电预诊断、精准投药、能源管理四大专家系统支撑下，延长了设备使用寿命，提高了用电监控，提升了投药精准度，节省了药耗及能耗。



本次入选，是对深圳龙华水务公司智慧厂站建设成果的充分肯定，未来将继续创新，探索推动供水生产服务全流程集中监控值守，以全业务数字化、智慧化管控推动公司高质量发展，并为行业提供更多值得借鉴的实践经验。


深圳水务科技公司愿助力深圳龙华水务公司水生产管控全业务数字化、智慧化管控，未来将继续夯基础、拓思路、高赋能水务智慧发展新篇章。
